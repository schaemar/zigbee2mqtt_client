# zigbee2mqtt_client

Clone the sources
```shell
git clone https://github.com/schaemar/zigbee2mqtt_client
```
Prepare the virtual environment

```shell
cd zigbee2mqtt_client
python3 -m venv <env_name>
source <env_name>/bin/activate
pip install -r path/to/requirements.txt
```
Run the client

```
python3 client.py
